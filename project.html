Este projeto consistia no desenvolvimento da Identidade Visual do evento CCDM, dando ênfase  ao digital. Para resolver o enunciado projetámos uma tipografia generativa que ganha forma em ambientes digitais como a web. Cada letra é construída sobre uma circunferência e, através da ligação de pontos nela definidos, criam-se linhas que originam a forma final. Cada um desses pontos varia entre duas a três unidades de maneira a criar diferentes formas para cada letra. Este projeto foi escolhido como imagem visual da edição 2023 do Ciclo (CCDM).
